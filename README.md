## 📖 Sobre o projeto

O **SKYREAD** é um site de previsão do tempo criado como projeto de aprendizado. Toda a jornada de desenvolvimento foi construída com o auxílio de IA (Claude - Anthropic), enquanto aprendia os fundamentos do desenvolvimento web na prática.

> 💡 *"Aprendo enquanto construo — cada linha de código é uma nova lição."*

---

## ✨ Funcionalidades

- [x] Busca de clima por cidade
- [x] Exibição da temperatura atual
- [x] Previsão para os próximos dias
- [x] Ícones dinâmicos de acordo com o clima
- [x] Design responsivo para mobile e desktop

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura das páginas |
| CSS3 | Estilização e layout |
| JavaScript | Lógica e consumo de API |
| API de clima | Dados meteorológicos em tempo real |
